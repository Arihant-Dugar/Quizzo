-> This is an Quiz App.

-> This app contains 5 Ques which are Hardcoded Static ques.

-> This app implements the Fragments and Navigation to switch between fragment and fragment
activity and fragment etc.

-> Fragments are used in this app to understand the flow of fragments better and they are reusable
light-weight in comparison to Activites.

-> Also there is a data model which holds the blueprint how the each ques should look-like.

-> Also implemented splash screen for better app presentation.
